This design was successfully implemented on a pre-release of the
"MAX 10 FPGA Evaluation Kit":
  https://www.altera.com/products/boards_and_kits/dev-kits/altera/kit-max-10-evaluation.html

Linux:
  Create ../quartus_env.sh (sample in ../quartus_env.sh.sample)
  Run in this directory from a bash shell
    ./0_create_quartus2.sh
  The created project (suitable for compilation with the Quartus II software)
  will be created in a sub-directory which is named after the Quartus II version you use.
